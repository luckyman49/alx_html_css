# Headphones Landing Page

## Description
This project is a responsive landing page for a headphones brand.  
Built with **HTML and CSS only**, based on a Figma design provided by ALX.

## Requirements
- No external CSS frameworks (Bootstrap, Tailwind, etc.)
- No JavaScript
- Fonts: Source Sans Pro and Spin Cycle OT
