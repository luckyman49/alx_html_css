# CSS Advanced Project

## Objectives
This project builds on the *HTML Advanced* project.  
The goal is to focus on *CSS styling* and make the page look like the design provided in Figma.

## Designer File
You can view the design here:  
[Page in Figma](https://www.figma.com/proto/R5f28Y8T0AWuUJMDcGdW0w/Homepage--Copy--in-alx-figma?node-id=3558-0&t=yRGlxv3XFE6K9kp1-1)

## Assets
Images and logos used in this project were exported directly from the Figma file.

## Notes
- Fonts used: Source Sans Pro and Spin Cycle OT.  
- Some values in Figma are floats; rounded values were used in CSS.  
- Semantic HTML structure was followed from the HTML Advanced project.  
- CSS Flexbox and responsive design principles were applied.
